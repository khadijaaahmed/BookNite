# BookNite
Book your stay where ever you want ....

Backend Architecture: Engineered robust RESTful APIs using Node.js, Express, and MongoDB. Implemented secure JWT authentication and developed real-time pricing calculations to provide dynamic quotes based on user input.

Frontend Development: Built a responsive and dynamic user interface with React.js. Features include advanced property discovery, real-time filtering, and an interactive booking workflow that enhances the user experience.

Core Functionality: Executed full CRUD operations for property and user management. Integrated secure image uploads, server-side date-based availability validation, and distinct workflows for dual-role user profiles (hosts and guests).
